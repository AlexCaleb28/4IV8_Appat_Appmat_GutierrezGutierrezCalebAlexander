Elementos Aprendidos

Durante este periodo, aprendí varias cosas importantes para el desarrollo y diseño web. Pude estructurar y darle diseño a páginas usando HTML y CSS para que se vean bien y se adapten a diferentes pantallas. También practiqué la lógica de programación con Java y agregué interactividad a las páginas con JavaScript. Lo principal que aprendí y apliqué fue armar operaciones CRUD (Crear, Leer, Actualizar y Eliminar), conectando formularios y tablas para poder gestionar información completa.
